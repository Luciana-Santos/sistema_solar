## Iniciando o projeto

Passo a passo de como rodar o projeto localmente.

### Instalação

1. Clone o repositório
   ```sh
   gh repo clone Luciana-Santos/Lucianadss-portfolio
   ```
2. Instale as dependências do projeto
   ```sh
   npm install
   ```
3. Inicie o servidor local
   ```js
   npm start
   ```
